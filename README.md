# Getting Ready for Physics Class

A command-line Python application designed to automate fundamental physics calculations for students and educators. This project showcases the clean utilization of Python functions, positional and keyword arguments, string formatting, and mathematical operations.

## Features

- **Temperature Conversion:** Seamlessly convert temperatures between Fahrenheit and Celsius.
- **Force Calculation:** Compute force (in Newtons) using mass and acceleration ($F = ma$).
- **Energy Estimation:** Calculate energy (in Joules) using Einstein's mass-energy equivalence formula ($E = mc^2$), utilizing a default speed of light constant.
- **Work Computation:** Measure physical work (in Joules) by chaining force calculations across a specified distance ($W = F \cdot d$).

## Getting Started

### Prerequisites

You only need Python 3.x installed on your machine to run this script.

### Running the Script

1. Clone this repository or download the `script.py` file.
2. Open your terminal or command prompt.
3. Run the script using the following command:
